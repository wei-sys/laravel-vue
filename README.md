## Contributing
```bash
# Clone the repository
git clone https://github.com/wei-sys/laravel-vue.git

# Navigate to the cloned directory
cd laravel-vue

# Install the dependencies
composer install

# Create a database
php artisan migrate

# Install the dependencies
npm i

# build the frontend
npm run build

# http://localhost/admin/#/dashboard
```
